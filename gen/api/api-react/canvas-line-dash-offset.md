canvasContext.lineDashOffset
============================

> 基础库 1.9.90 开始支持，低版本需做[兼容处理](https://developers.weixin.qq.com/miniprogram/dev/framework/compatibility.html)。

### 定义

设置虚线偏移量的属性

### 语法

    canvasContext.lineDashOffset = value
    

### 参数

  属性值  |  类型     |  说明         
----------|-----------|---------------
  value   |  Number   |偏移量，初始值为 0
